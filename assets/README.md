# Assets

Upload the signature images here with these exact filenames:

- `headshot.png` - square headshot, ideally 256 x 256 px or larger
- `facebook.png` - Facebook icon, ideally 60 x 60 px
- `instagram.png` - Instagram icon, ideally 60 x 60 px
- `linkedin.png` - LinkedIn icon, ideally 60 x 60 px
- `zillow.png` - Zillow icon, ideally 72 x 60 px or similar

After GitHub Pages is enabled, public URLs will use this pattern:

```text
https://joepinerealtor.github.io/JPRSigniture/assets/headshot.png
```

The main signature file is `signature.html`.
